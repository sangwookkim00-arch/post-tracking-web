# 📮 우체국 등기 배송 조회 서비스

우체국 공개 API를 활용하여 등기 배송 상태를 실시간으로 조회할 수 있는 Streamlit 웹 애플리케이션입니다.

## ✨ 주요 기능

- 📁 **엑셀 파일 업로드**: 등기번호가 포함된 엑셀 파일을 업로드
- 🔍 **실시간 조회**: 우체국 API를 통한 배송 상태 실시간 조회
- 📊 **진행률 표시**: 조회 진행 상황을 시각적으로 표시
- 💾 **결과 다운로드**: 조회 결과를 엑셀 파일로 다운로드
- 🎨 **현대적인 UI**: 그라디언트와 카드 스타일의 세련된 디자인

## 🚀 설치 및 실행

### 1. 저장소 클론
```bash
git clone https://github.com/sangwookkim00-arch/post-tracking-web.git
cd post-tracking-web
```

### 2. 패키지 설치
```bash
pip install -r requirements.txt
```

### 3. 앱 실행
```bash
streamlit run app.py
```

브라우저가 자동으로 열리며 `http://localhost:8501`에서 앱을 사용할 수 있습니다.

## 📋 사용 방법

1. **엑셀 파일 준비**: 등기번호가 포함된 엑셀 파일(.xlsx)을 준비합니다.
2. **파일 업로드**: 웹 페이지에서 파일을 업로드합니다.
3. **컬럼 선택**: 등기번호가 포함된 컬럼을 선택합니다.
4. **조회 시작**: '조회 시작하기' 버튼을 클릭합니다.
5. **결과 확인**: 실시간으로 조회 결과를 확인합니다.
6. **다운로드**: 조회 완료 후 결과를 엑셀 파일로 다운로드합니다.

## 🛠️ 기술 스택

- **Streamlit**: 웹 애플리케이션 프레임워크
- **Pandas**: 데이터 처리 및 엑셀 파일 핸들링
- **Requests**: HTTP API 호출
- **XML ElementTree**: XML 응답 파싱
- **XlsxWriter/OpenPyXL**: 엑셀 파일 읽기/쓰기

## 📦 필요한 패키지

```
streamlit

```
